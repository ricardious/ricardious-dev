# ricardious-dev

Portfolio personal enfocado en dirección visual, motion y experiencias web inmersivas.

Su lenguaje visual toma referencias del tipo de sensibilidad editorial visto en portfolios como el de Cathy Dolle.

## Stack

- Astro 6
- React 19
- Tailwind CSS 4
- Three.js + React Three Fiber + Drei
- GSAP
- TypeScript

## Desarrollo

```sh
pnpm install
pnpm dev
pnpm build
pnpm preview
```
